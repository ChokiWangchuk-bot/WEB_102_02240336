# Practical 2: Weather API Dashboard
## What is this project?
This project is a Weather API Dashboard developed with HTML, CSS, and JavaScript.
It showcases the use of RESTful APIs by implementing all four core HTTP methods: GET, POST, PUT, and DELETE.
## Features Implemented
- A dashboard with a tabbed interface divided into four sections
- A GET tab that retrieves current weather information based on city name using the OpenWeatherMap API
- A POST tab for adding a new location to a simulated REST backend
- A Saved Locations tab that allows users to view, update, or remove stored locations
- An API Info tab that displays details from the most recent API request
## Technologies Used
- HTML and CSS – For page structure and visual design
- JavaScript (using the Fetch API) – To handle HTTP requests
- OpenWeatherMap API – To obtain live weather data
- JSONPlaceholder API – To emulate create, update, and delete operations
